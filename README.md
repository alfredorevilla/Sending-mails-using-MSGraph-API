# Sending mails using (MS) Graph API

## Requirements
[Download and install .NET Core 3.1 runtime or SDK](https://dotnet.microsoft.com/download/dotnet-core/3.1)


## Setup
Open Program.cs and replace all `<value`> instances found between double quotes.

## Run

From command prompt run:

`dot net run`
